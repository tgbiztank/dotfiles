# Installation

```
cd ~
git clone https://github.com/goMb1z/dotfiles.git
cd dotfiles
sudo chmod +x i1.sh
./i1.sh
rm -rf ~/dotfiles
```
### Install NodeJS for CocVim
```
sudo pacman -S nodejs
```
if command does not work please use this
```
sudo -i
curl -sL install-node.vercel.app/lts | bash
exit
exit
```
